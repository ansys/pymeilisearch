Getting started
###############

This guideline presents the different steps on how to setup the environment to
use ``pymeilisearch``.

Access to a local or remote installation of `meilisearch`_ is required before
using ``pymeilisearch``.

.. _meilisearch: https://www.meilisearch.com/
